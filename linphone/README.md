# Linphone 

This built a version of Linphone android app for YAMIR.

Default make target will

- Download source code from git://git.linphone.org 
- Apply patch files
- Build the android Linphone app.

Note git://git.linphone.org is no longer available.

As of 2020 Belledonne Communications deprecated use of it git archives.

This has effectively ended support for custom builds like YAMIR.
